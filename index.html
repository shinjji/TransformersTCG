📋 Setup Guide — Google Cloud API Key
1
Create a Google Cloud Project
Go to console.cloud.google.com and sign in with your Google account. Click New Project, give it a name like "TCG Collection", and create it.
2
Enable the Google Sheets API
In your project, go to APIs & Services → Library. Search for "Google Sheets API" and click Enable.
3
Create an API Key
Go to APIs & Services → Credentials. Click + Create Credentials → API Key. Copy the key and paste it into the field above.

⚠️ Important: Click Restrict Key and limit it to the Google Sheets API only for security.
4
Make Your Sheet "Anyone with the link can view"
Open your Google Sheet. Click Share → General Access → Anyone with the link → Viewer. This is required for API key access (read). For write access (adding/editing cards), see step 5.
5
Enable Write Access via Service Account (for adding/editing cards)
For write access, you need a Service Account instead of just an API key:
— Go to Credentials → + Create Credentials → Service Account
— Once created, go to it and click Keys → Add Key → JSON — download the file
— In your Google Sheet, click Share and add the service account email (ends in @...iam.gserviceaccount.com) as an Editor

Then paste the entire JSON key file content into the field below:
